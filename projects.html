Visual Arduino project with Processing
Maastricht, The Netherlands

The project is created through the concept of generative art. The project is divided in an analog and a digital interactive design. The two tools are connected with each other.

The main topic of this project was about meditation. After pressing the conductive tape which is connected with each square, the user will be shown a pattern on the screen of the laptop. By pressing the white dot on the black carton board the system will randomly insert words 'of wisdom' on the screen, which are related with the book; Tao Te Ching by Lao Tzu.

This project was created with the help of Processing, an Arduino and a MPR121 sensor at the first months of the Corona outbreak.

Used skills:

Storytelling - Photoshop - Processing - Arduino

In a school assignment we had to create a code-workshop for children in secondaryschool. These are children between the age of 11-13.
